# Estructura general de pagina web

## Lista de archivos

- index.html
- public
    - images
- js
- Styles




en el  archivo html la etiquet `title`
establece el titulo de la pagina

```html

<title> nombre pagina </title>

```

en el  archivo css la etiquet `title`
establece el titulo de la pagina

```css

<title> nombre css </title>

```
